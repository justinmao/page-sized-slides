page-sized-slides.js

Intuitive & elegant.

Navigate by scrolling or using up/down arrow keys (or by swiping up/down on mobile).

Transitions look and feel fluid and natural thanks to lethargy, smooth-scroll, and a native implementation of mobile gesture recognition.

> lethargy
> smooth-scroll

Simple & easy to use.

Simply include <script src="page-sized-slides.js"></script> in your document header and add class="slide" to each page element tag to get started*.

May I suggest using <section> semantic tags?
*Each page must have an id attribute.

Powerful & customizable.

Add navigation tools using scrollToPage() or scrollToPageNumber().

Control exactly how page transitions should behave using built-in functions like setPageInit() and setPageDeinit().

> Click here for more details + documentation!

Lightweight.

At only 9kb minified, load-time is minimally impacted.
