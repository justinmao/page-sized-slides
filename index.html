page-sized-slides.js

Elegant & smooth.

Transitions and controls look and feel fluid and natural thanks to lethargy and smooth-scroll.

> lethargy
> smooth-scroll

Simple & easy to use.

Simply include <script src="page-sized-slides.js"></script> in your document header and add class="slide" to each page element tag to get started.

May I suggest using <section class="slide"> semantic tags?

Powerful & customizable.

Add navigation tools using scrollToPage() or scrollToPageNumber().

Control exactly how page transitions should behave using built-in functions like setPageInit() and setPageDeinit().
